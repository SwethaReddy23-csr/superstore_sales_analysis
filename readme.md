# Superstore Sales Analysis

- ✅ Power Pivot & Data Model
- 📊 Interactive Dashboard with Slicers
- 📈 PivotTables and Charts
- 🤖 Excel Macros
- 🧮 What-If Analysis & Scenario Manager
- 🔍 Outlier Detection & Data Cleaning
